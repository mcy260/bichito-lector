Conectar con GitHub

Los cambios se guardan directo en el repositorio y el sitio se republica solo. Necesitás un token de GitHub (se crea una sola vez y queda guardado en este navegador):

  1. Entrá a github.com/settings/personal-access-tokens/new
  2. Nombre: admin bichito · Expiración: la más larga
  3. En Repository access elegí Only select repositories → bichito-lector
  4. En Permissions → Contents elegí Read and write
  5. Generá el token y pegalo acá:
